## API

### Switch

| 参数           | 说明             | 类型                   | 默认值 |
| -------------- | ---------------- | ---------------------- | ------ |
| checked        | 指定当前是否打开 | boolean                | false  |
| defaultChecked | 初始是否打开     | boolean                | false  |
| loading        | 加载状态         | boolean                | false  |
| disabled       | 禁用状态         | boolean                | false  |
| onChange       | 变化时回调函数   | (val: boolean) => void | -      |
