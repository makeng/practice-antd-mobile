# API

| 属性        | 说明           | 类型                                                  | 默认值    |
| ----------- | -------------- | ----------------------------------------------------- | --------- |
| status      | 默认错误类型   | 'default' \| 'disconnected' \| 'empty' \| 'forbidden' | 'default' |
| title       | 标题           | ReactNode                                             | -         |
| description | 描述           | ReactNode                                             | -         |
| image       | 图片           | string \| ReactElement                                | -         |
| fullPage    | 是否为整页异常 | boolean                                               | false     |
