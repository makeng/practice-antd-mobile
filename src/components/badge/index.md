## API

### Badge

| 属性    | 说明                                       | 类型             | 默认值  |
| ------- | ------------------------------------------ | ---------------- | ------- |
| content | 徽标内容，不传即默认的红色小圆点           | React.ReactNode  | -       |
| color   | 徽标背景色                                 | string           | #FF411C |
| offest  | 徽标相对于默认位置的偏移量[x, y]， 单位 px | [number, number] | [0, 0]  |
