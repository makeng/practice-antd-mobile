## API

### ProgressCircle

| 参数        | 说明              | 类型              | 默认值    |
| ----------- | ----------------- | ----------------- | --------- |
| percent     | 百分比            | number            | 0         |
| strokeColor | 进度条颜色        | string            | `#1677FF` |
| strokeWidth | 线条宽度，单位 px | number            | 6         |
| size        | 画布宽高，单位 px | number            | 50        |
| children    | 自定义信息        | `React.ReactNode` | -         |
