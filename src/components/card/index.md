## API

| 属性          | 说明                | 类型       | 默认值 |
| ------------- | ------------------- | ---------- | ------ |
| title         | header 左边区域     | ReactNode  | -      |
| extra         | header 右边区域     | ReactNode  | -      |
| onClick       | 卡片点击事件        | () => void | -      |
| onHeaderClick | Header 区域点击事件 | () => void | -      |
| onBodyClick   | Body 区域点击事件   | () => void | -      |
